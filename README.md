# Topcon GUI Simulator

A web-based eye refraction testing simulator built with React and TypeScript that mimics the Topcon CV-5000 phoropter.

## Project Structure

```
src/
├── components/      # React components (Phase 2-4)
├── context/         # Context API for state management
├── hooks/           # Custom hooks
├── types/           # TypeScript type definitions
├── utils/           # Utility functions
├── styles/          # CSS and Tailwind styles
└── main.tsx         # Entry point
```

## Features (Phase 1 Completed)

- ✅ Project setup with React 18 + TypeScript + Vite
- ✅ Tailwind CSS configured
- ✅ Context API setup for state management
- ✅ Type definitions created
- ⏳ Phase 2: Refraction Table implementation
- ⏳ Phase 3: Control components (PD, Occlusion, Eye modes)

## Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Running Development Server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

## Technologies Used

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Context API** - State management
