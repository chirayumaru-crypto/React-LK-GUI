# One Simulator

## Overview
One Simulator is a next-generation optometry interface designed with a focus on premium aesthetics and futuristic usability.

## Features
- **Premium Glassmorphism Design**: High-end visual interface with deep space blues and neon accents.
- **Digital Refraction Control**: Precision inputs for Sphere, Cylinder, Axis, and Add.
- **Interactive Chart System**: Dynamic chart selection and viewing area.
- **Responsive Layout**: Adaptive glass panels for optimal viewing.

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS (Custom Premium Config)
- Lucide React Icons

## Getting Started
1. `npm install`
2. `npm run dev`
